selector cheat sheet

前提条件

  • 表中の "a", "b" は要素を表す。セレクタによっては"a"がアンカー<a>の意味と解釈できる場合があるが、あくまでも代理の文字。
  • チートシート作者が覚えにくいと思っているセレクタのみ。すべてのセレクタを網羅しているわけではない。
  • 赤文字部分が各セレクタの特徴的な記述部分。

汎用セレクタ

selector name description browser
a > b 子セレクタ aの要素の直下にbの要素があった場合選択される。
a + b 隣接セレクタ aの要素の次に出現する要素を選択する。
a * ユニバーサルセレクタ *単体で全ての要素を選択。要素名 (スペース) * とすることで要素名で指定した要素内の全ての要素を選択する。
a ~ b 間接セレクタ aの要素以降に出現する要素を選択する。

擬似クラス

selector name description browser
a:first-child first-child擬似クラス 複数あるaの要素の最初の要素を選択。aを含む親要素が複数ある場合、各親要素の中の最初に表示されるaが選択される。
a:not(b) 否定擬似クラス 複数あるa要素の中から除外対象を:not()の括弧内に記述。括弧で指定された要素b以外を選択する。
a:empty empty擬似クラス 子要素やテキストを含まないa要素を選択。
a:nth-child(num) nth-child擬似クラス 複数あるa要素から任意の番号の要素を選択する。番号は括弧内に記述。カウントは1から。
a:last-child last-child擬似クラス 複数あるa要素から、その要素の最後の要素を選択する。
a b:only-child only-child擬似クラス aの要素内にb要素が1つだけ含まれる場合にb要素を選択する。

属性セレクタ

selector name description browser
[attribute] - attributeで指定した属性([ ]で指定した属性)を持つ要素を選択する。
[attribute='value'] - attributeで指定した属性がvalueで指定する値を持つ場合に選択。
[attribute!='value'] - attributeで指定した属性がvalueで指定する値を持たない要素を選択する。
[attribute^='value'] - attributeで指定した属性がvalueで指定する文字列で始まっている要素を選択する。
[attribute$='value'] - attributeで指定した属性がvalueで指定する文字列で終わっている要素を選択する。
[attribute*='value'] - attributeで指定した属性がvalueで指定する文字列を含んでいる要素を選択する。

jQuery独自フィルター

selector name description browser
a:first firstフィルター a要素の1番目の要素のみ選択。
a:last lastフィルター a要素の最後の要素のみ選択。
a:even evenフィルター a要素の偶数番目の要素を選択。
a:odd oddフィルター a要素の奇数番目の要素を選択。
a:eq(num) eqフィルター (num)で指定された順番のa要素を選択する。(num)は0からカウントする。
a:gt(num) gtフィルター (num)で指定された数字以上の順番のa要素を選択する。(num)は0からカウントする。
a:lt(num) ltフィルター (num)で指定された数字以下の順番のa要素を選択する。(num)は0からカウントする。
a:header headerフィルター aにIdやClassを指定するとそのId,Class内のh1～h6を選択する。
a:contains(str) containsフィルター (str)で指定した文字列を含むa要素を選択。
a:has(b) hasフィルター (b)で指定した要素を文字列を含むa要素を選択。
a:parent parentフィルター 何らかの子要素やテキストを含むa要素を選択。
